# Petodo 项目进度表

> 状态说明：已完成 / 部分完成 / 未完成 / 有问题，需要修复

## 总体判断

当前项目已经完成后端主干、桌宠窗口基础联动、主窗口新版展示界面和积分商店主流程：FastAPI 后端可以稳定提供番茄钟、桌宠状态、喂食、饥饿值、本地保存、基础专注记录、积分和总状态查询；主窗口已重设计为三栏学习日志式界面，并提供首页番茄钟、待办清单、补给商店、专注记录、设置和桌宠状态卡的基础前端交互。主窗口投喂和补给商店会先检查积分，兑换成功后扣除积分并触发桌宠 eating 和 finished_eating。主窗口倒计时已改为逐秒平滑显示，专注完成后会弹出“恭喜计时完成”提示和烟花效果，宠物会先 happy 再进入 rest。

项目总进度估计：约 84%。
